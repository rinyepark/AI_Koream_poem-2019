## 1. Duration
- 2019.03 ~ 2019.06

## 2. Skills
- Python, ML(RNN, LSTM)
- Data Crawling, Data Preprocessing, Modeling

## 3. Contents
- LSTM을 활용하여 한국 시를 학습시킨 후 한국어로 된 시를 짓는 AI를 개발함
- 데이터 크롤링을 통해 시 데이터 수집
* 기존에 개발된 LSTM모델 참조

 1) 크롤링을 통한 한국 시 데이터 수집 및 전처리
